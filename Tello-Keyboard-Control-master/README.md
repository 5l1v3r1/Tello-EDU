The UDP Port was already set, just connect to the wifi of the Tello, and by using the keys:

"w" - forward <br>
"a" - left <br>
"s" - back <br>
"d" - right <br>
"->" - (arrow key right) clockwise turn (about 15-25 degrees) <br>
"<-" - (arrow key left) counter clock wise (about 15-25 degrees) <br>
"q" - land mode
